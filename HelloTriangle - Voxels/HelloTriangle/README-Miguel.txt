Save/Load: 
  Voxels salvos após apertar ESC e fechar o programa
  Para carregar um arquivo, mude o nome do arquivo na linha 155

Indicador:
  Linha 275 começa a implementação do indicador de cores que esta posicionado ao lado da grid de voxels.

Controls:

- CURSOR
  * NP 8 == UP key == z movement(+)
  * NP 2 == DOWN key == z movement(-)
  * NP 4 == LEFT key == x movement(-)
  * NP 6 == RIGHT key == x movement(+)
  * NP 7 == I key == y movement(+)
  * NP 1 == K key == y momvement(-)
  * NP 9 == Cycle colors (+)
  * NP 3 == Cycle colors (-)
  * SPACE == Add/remove voxel

- CAMERA
  * V key == Top-down view, also resets camera 	     to starting position
  * Q key == Moves camera up
  * E key == Moves camera down  
  * WASD == Controls camera relative to where 		    mouse is pointing
